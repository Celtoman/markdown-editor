# Interactive Markdown Editor

Современный онлайн-редактор Markdown с живым предпросмотром, статистикой, структурой документа и экспортом в несколько форматов.

## Возможности

- Живой предпросмотр Markdown в реальном времени.
- Поддержка GFM через `marked`.
- Санитизация HTML через `DOMPurify`.
- Подсветка кода (`highlight.js` + `marked-highlight`).
- Экспорт в `.md`, `.html` и `.pdf`.
- Режимы работы: разделение, только предпросмотр, фокус.
- Полноэкранный предпросмотр.
- Синхронизация скролла редактора и предпросмотра.
- Статистика текста и навигация по заголовкам (H1-H4).
- Темы: светлая, темная, системная (с сохранением выбора в cookie).
- Адаптивный интерфейс для desktop и mobile.

## Технологии

- React 19
- TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Marked + DOMPurify

## Быстрый старт

Требования:

- Node.js 18+
- npm

Установка и запуск:

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу:

```text
http://localhost:3000
```

## Скрипты

| Скрипт | Назначение |
| --- | --- |
| `npm run dev` | Запуск проекта в режиме разработки |
| `npm run build` | Production-сборка в `dist/` |
| `npm run preview` | Локальный предпросмотр production-сборки |
